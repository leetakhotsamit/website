BIOL469: Genomics

Focused on learning the linux command line. Experienced how to run the BLAST command line and the prokka command line. Became familiar with tools such as FASTQC, Velvet, Abyss, Tablet, Artemis, Bandage, Muave, BWA, samtools and bcf tools. The course incorporated a project to which my team created a commandline that performed Genome Assembly, Annotation and Functional comparison tasks all at once. The pipeline that my team created was comparable to prokka, which is a tool to annotate bacterial, archaeal and viral genomes quickly and produces standards-compliant output files.

Project: Creating a comparative genomics pipeline that was similar to prokka. It takes Illumina raw sequence reads (either HiSeq or MiSeq platform), assembles genomes, predicts genes, annotates predicted genes, and performs basic comparisons. For quality control, the tools trimmomatic and FastQC were incorporated. The tool used for genome assembly was SPAdes. For gene prediction, we implemented tools such as Prodigal and FragGeneScan. We were then able to align the genes using the Diamond tool. Advantages with using our pipeline was the fact that it was able to compare more genes compares to those compared using prokka.
